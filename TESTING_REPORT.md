## 📐 响应式设计详情

### 断点策略 (Mobile First)

```css
/* 基准: 移动端 (320px - 767px) */
默认样式

/* 平板: 768px - 1023px */
@media (min-width: 768px) {
  - 导航切换为水平布局
  - Hero 显示双背景图
  - Testimonials 水平布局
  - CTA 按钮调整尺寸
}

/* 小桌面: 1024px - 1439px */
@media (min-width: 1024px) {
  - Reasons 复杂定位布局
  - Instructors 横向轮播
}

/* 大桌面: 1440px+ */
@media (min-width: 1440px) {
  - 最大宽度限制
  - Journey 显示背景文字
  - Support 三列阶梯布局
}
```

### 流体排版示例

```css
/* 使用 clamp() 实现流体缩放 */
.hero__title {
  font-size: clamp(2.5rem, 10vw, 4.95rem);
}

.section-title {
  font-size: clamp(1.5rem, 5vw, 2.25rem);
}

.testimonial__text {
  font-size: clamp(0.875rem, 3vw, 1rem);
}
```
