# HTML页面拆分

## 📁 项目结构

```
tosho-html/
├── index.html              # 主页面（单页应用）
├── css/
│   └── style.css           # 主样式文件（Mobile First）
├── js/
│   └── main.js             # 交互脚本（导航、轮播等）
├── images/                 # 图片资源目录
│   ├── logo-*.png          # Logo 图片
│   ├── hero-*.png/svg      # Hero 区域背景和装饰
│   ├── about-*.jpg/svg     # About 区域图片
│   ├── testimonial-*.jpg   # 学员评价照片
│   ├── reason-*.jpg        # 选择理由配图
│   ├── support-*.jpg       # 案件支持配图
│   ├── instructor-*.jpg    # 讲师照片
│   ├── icon-*.svg          # 功能图标
│   └── *-bg.svg/png        # 背景装饰图
├── README.md               # 项目说明文档
```
